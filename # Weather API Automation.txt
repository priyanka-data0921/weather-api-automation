# Weather API Automation

This Python project retrieves real-time weather data using the OpenWeather API and sends desktop notifications automatically.

## Features
- Fetches weather data using REST API
- Runs automatically on a schedule
- Shows desktop notifications
- Parses JSON API responses

## Technologies Used
- Python
- Requests
- Schedule
- Plyer
- OpenWeather API

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Open weather_script.py and replace:
API_KEY = "YOUR_API_KEY"

with your real OpenWeather API key.

3. Run the script:
python weather_script.py

## Example Use Case
This project demonstrates API integration, scheduling, JSON parsing, and desktop notification automation in Python.
